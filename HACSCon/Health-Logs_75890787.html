  • Following commands at https://github.com/UMD-ACES/Honeypot_Project/wiki/Health-Logs to create health log

Commands to capture data on proxmox VM:

  • date=date

  • ram=awk '/^Mem/ {print $7}' <(free -m)
    disk=awk '/^\/dev/ {print $4/1024}' <(df /dev/mapper/pve-root)
    load=uptime | cut -d"," -f 6
    netin=awk '{print $2/1024}' <(cat /proc/net/dev | grep "enp4s2")
    netout=awk '{print $10/1024}' <(cat /proc/net/dev | grep "enp4s2")

Commands to capture data on Containers:

  • ram=awk '/^Mem/ {print $7}' <(/usr/sbin/pct exec $ct_num free)
    disk=awk '/^\/dev/ {print $4/1024}' <(df /var/lib/lxc/$ct_num/rootfs)
    load=/usr/sbin/pct exec $ct_num uptime | cut -d"," -f 5
    netin=awk '{print $3/1024}' <(lxc-info -H $ct_num | grep "RX bytes")
    netout=awk '{print $3/1024}' <(lxc-info -H $ct_num | grep "TX bytes")

  • Grabbing next row value from row.txt, edit it at the end of script to add one

  • Bash script to collect data, data is sent to a python code: python log.py <data separated by ,>

  • Crontab -e

    • 00 * * * * /root/Health_Logs/health_log > /root/Health_Logs/foo.txt

    • Runs every hour

    • note: everything must be absolute paths for crontab to work, even files referenced within the scripts and python code
